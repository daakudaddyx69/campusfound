# CampusFound - Campus Lost & Found Utility Portal

CampusFound is a centralized, secure repository and communication platform designed to help students and faculty recover lost personal items and return logged found items on campus.

## Key Features

- **Consolidated Repositories**: Separate, real-time feeds for active **Lost Items** and **Found Items**.
- **Secure Claims & Matching**: Simple options to log lost or found statuses with precise campus location markings.
- **Integrated Message Center**: In-app private chats connecting claimants directly to item finders without exposing personal contact details publicly.
- **Smart Management**: Toggle resolved items or archives, and inspect detailed visual status badges.
- **Mobile-First Layout**: Fully responsive, dark brutalist UI built to be fast and accessible on smartphones and tablets alike.

## Tech Stack

- **Frontend**: React (v18+) with Vite, TypeScript, and Tailwind CSS.
- **Analytics & Motion**: Framer Motion for clean status transitions.
- **Backend & Database**: Firebase Firestore for persistence and private messaging threads.
- **Authentication**: secure Google Workspace/Student Google Sign-In.
